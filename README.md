 Portfolio Maker
A modern, responsive web application that allows users to create, preview, and download a professional personal portfolio without writing any code.
Portfolio Maker provides an intuitive interface where users can enter their personal information, skills, education, projects, and certifications while instantly previewing the final portfolio in multiple templates.

Built With
HTML5
CSS3
JavaScript (ES6)
Local Storage API
FileReader API
Responsive CSS Grid & Flexbox
